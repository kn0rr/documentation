# Math Object

Some Examples:
````js
Math.PI // 3.141592653589793

//Absolute value:
Math.abs(-432) //432

// Raises 4 to the 5th power:
Math.pw(4,5) //1024

//Removes decimal 
Math.floor(3.9999) //3

// Returns random decimal
Math.random()

//Returns random decimal between 0 and 6
Math.random()*6

// Returns value between 0 and 6 with removed decimal!
Math.floor(Math.random()*6)
````
